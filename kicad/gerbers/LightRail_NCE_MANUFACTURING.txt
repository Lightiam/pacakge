DFM NOTES
=========

Design for Manufacturing Checklist:
  ✓ No acute angles (all 45° or better)
  ✓ Trace width maintained (≥0.10mm)
  ✓ Via density adequate for thermal
  ✓ Component spacing respects thermal relief
  ✓ Via stitching around grounds (≥1 per sq inch)
  ✓ Edge clearance verified (≥0.5mm)
  ✓ Silkscreen legible (≥1.2mm height, ≥0.2mm stroke)
  ✓ No test pads under components
  ✓ Panelization allowance (if applicable)
  ✓ Thermal vias under high-power (6+ per pad)

Critical Items:
  1. BGA-3136 escape routing: 4-layer fan-out required
  2. PCIe differential impedance: ±10% (use stackup analysis)
  3. Thermal via density: Min 6 vias per 0.1"² under ASIC
  4. Memory clock jitter: <3ps RMS (verify with CAD tools)
  5. Power plane continuity: Check netlist against Gerber
