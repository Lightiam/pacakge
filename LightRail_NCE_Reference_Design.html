1. System Overview & Architecture

1.1 Product Description

The LightRail AI NCE (Neural Computing Engine) Motherboard is a high-performance compute accelerator designed for enterprise AI inference and training applications. This reference design provides complete specifications for PCB design, manufacturing, and assembly.

1.2 Key Specifications

Computing Power: 1.5 TFLOPS FP32 @ 2.4 GHz
Memory Bandwidth: 256 GB/s aggregate HBM3
Total Memory: 192GB (12x 16GB HBM3 stacks)
Power Delivery: 225W typical / 275W peak
Operating Frequency: 2.4 GHz nominal
Form Factor: Dual-slot PCI Express x16
Dimensions: 267mm × 111mm × 12mm
PCB Layers: 10-layer controlled impedance

7.3 Fault Detection & Shutdown

Monitored Faults:
• Overvoltage (OVP): >1.0V on Vcore → Shut down PDM
• Overcurrent (OCP): >220A peak → Current limit engaged
• Overtemperature: >95°C junction → GPU reset + throttle
• Undervoltage (UVLO): <0.8V on Vcore → Emergency shutdown
• Power loss: All supplies <5V → Graceful shutdown

Graceful Shutdown Sequence:
1. Assert GPU reset immediately
2. Disable memory refresh
3. Stop clock generation
4. Release all enable signals
5. Capacitors discharge naturally
6. Total time: <500ms to safe state

8.2 On-Chip PLL & Clock Multiplier

PLL Architecture (Integrated in LR-GEN3-NPU):

Input Stage:
• Reference frequency: 27 MHz ±50 ppm
• Phase detector: ±60 dBc phase noise
• Reference divider: ÷1 (no reduction)

Multiplier Stage:
• Multiplication factor: x88.9
• Output frequency: 2.4 GHz (27 × 88.9 ÷ 1)
• VCO range: 2.2 - 2.6 GHz
• Feedback divider: ÷32 (for lock range)

Output Characteristics:
• Frequency accuracy: ±0.1%
• Jitter (period): <2 ps RMS
• Phase noise @ 1MHz offset: <-100 dBc/Hz
• Duty cycle: 50% ±5%
• Lock time: <10 microseconds

Programmable Features:
• Spread spectrum: Disabled (for simplicity)
• Loop bandwidth: Fixed (optimized for stability)
• Bypass mode: Emergency fallback (if needed)
